API de Cotação de Moedas

Este projeto consiste no desenvolvimento de uma API RESTful em **Python** (utilizando **FastAPI**) que fornece cotações de moedas em tempo real. A API consulta dados de uma fonte externa confiável e os disponibiliza de forma padronizada para integração com outras aplicações.

Tecnologias Utilizadas:

Python FastAPI Requests JSON Docker (opcional, mas bom para mencionar se usar)
